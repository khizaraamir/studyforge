# StudyForge

An AI-powered flashcard generator built with React, Node.js, and the Claude API.

## Features
- Paste any notes and generate flashcards instantly
- Flip-card study interface
- Track progress with "Got It" / "Study Again"
- Progress bar showing mastery percentage

## Tech Stack
- **Frontend:** React, Vite, Tailwind CSS
- **Backend:** Node.js, Express
- **AI:** Anthropic Claude API

## Running Locally
1. Clone the repo
2. `cd client && npm install && npm run dev`
3. `cd server && npm install && node index.js`
4. Add your Anthropic API key to `server/.env`
